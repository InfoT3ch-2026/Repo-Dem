## Description
This is a simple GUI calculator program that allows users to perform basic arithmetic operations.
The calculator supports addition, subtraction, multiplication, and division, and displays 
the current number and result in a textbox or label control.

## Features
-> Digital buttons for 0 through 9

-> Operator buttons: '+', '-', '*', '/', and '='

-> Clear button to reset all values

-> Display shows combined digits as a single number

-> Running total is updated according to the last operator clicked

-> Equal button calculates the final total and displays it

## How it works
1. User clicks digit buttoms -> number is displayed in the textbox.

2. Clicking an operator button adds the current value to the running total using the last operator.

3. If no operator had been clicked yet, the value is added by default.

4. Clicking the equal ('=') button computes the final result and display it.

5. Clicking the clear button resets all values to zero.

## Technologies/Tools used
